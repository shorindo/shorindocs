# プラグインの仕組み

1. プラグインはJavaScriptとして実装・実行される
2. プラグインの実行はイベント通知によって行われる
3. プラグインごとにどのイベントを処理対象とするかを決め、対象としたイベントのみが送られる
4. 異なるプラグインは同一スレッドで直列に動作するが、メモリ空間は独立して相互に干渉しない
